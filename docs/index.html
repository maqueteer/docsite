The Interface

  1. The Toolbar: The Toolbar provides quick and easy access to important tools and actions you‚Äôll use a lot in Maqueteer.

  2. The Canvas: The canvas is where you visualize, customize, or create from scratch your spread design. It shows the current selected spread.

  3. The Inspectors: In the Inspectors, you‚Äôll find settings for any element you have selected, or controls for the tool or feature you‚Äôre using:

    1. Properties: The properties inspector contains settings and controls for the current frames selection in the canvas or the spreads selection in the navigator.
    2. Text: The Text inspector allows you to change the text settings of the current selection.
    3. Image information: Show useful information of the current image selected in the canvas or in the image browser.
    4. Document Size: Settings of your album that you can change at any time. Also default settings for the new elements created such as frames or spreads.
    5. Layout Solutions: At any time, contains up to 100 layout solutions specially tailored for the images contained in the current spread.

  4. The Navigator: It contains all the spreads of your album. Here you can select one or more spreads, rearrange them and so on.

  5. The Image Browser: It contains all images imported in your project. Most of the time It's the starting point to add images to your album.

The Toolbar‚Äã

The Toolbar is where you‚Äôll find quick shortcuts to important tools and actions you‚Äôll use a lot in Maqueteer. Here‚Äôs a rundown of the toolbar:

  1. Add Spread: Lets you add a new blank spread in your album, at the end of the last selected spread in the navigator.
  2. Zoom: Change the percentage of zoom in the canvas or fit in the window automatically.
  3. Add to Favorites: Save the current layout in the canvas as a favorite so you can reuse it in this album or in another one in the future. (Go to layout solutions to know more about favorite layouts).
  4. Insert Frame, insert Text, Edit Frame, Edit Text: Give you quick access to common action for editing the frames in the canvas. (Find out more in Spreads)
  5. Export: Lets you export your album. (Find out more in Exporting)
  6. Sidebars: Easily show or collapse the navigator, inspector or image browser sidebars.

The Inspector‚Äã

In the Inspector, you‚Äôll find settings for any frames, images or spreads you‚Äôve selected, or controls for the tool you‚Äôre using. The Inspector enables these controls and settings based on what you have selected or what you‚Äôre doing.

To show or hide the inspectors, click the right button in the Sidebars control in the toolbar. Alternatively, use the main menu or their shortcut.

You can resize any inspector or collapse dragging its left edge.

The inspector consists of four tabs:

Properties‚Äã

The properties inspector contains settings and controls for the current selected frames in the canvas or the selected spreads in the navigator:

  1. Frame properties: These let you change the position, size, color, border and more of your currently selected frames.

  2. Image properties: If those frames contain images also let you change its zoom, rotation and opacity.

  3. Spread properties: With these controls you can change the gap and background of the spreads as well as flip them.

Album Settings‚Äã

Dimension settings of your album that you can change at any time. Also default settings for the new elements created such frames or spreads:

  1. Dimension properties: Let you change the size, cut/trim and safe areas of your album. You can also use a size preset or create a new one with the current settings. (find out more in Album Settings)

  2. Default properties: These are the properties that will be used any time you add a new spread or frame to your albums.

  1. Show the favorites solutions that match the number of frames in your canvas.

  2. Click to access the setting panel of the solutions.

  3. Up to 100 layout solutions. Click the one you wish and the layout will be applied to the current selected spread.

Image Browser‚Äã

The image browser contains all the images imported to your project.

  1. Click Import, to import more images to your project.

  2. Lets you sort your images in different ways:

  3. The filter button gives you the filter criteria available for your images. The options are dynamic and depends on the actual metadata of your images:

    You can filter for: usage, issues, labels, parent folders, keywords, rating and capture time. Inside of these categories each criterion shows the number of images that match. To filter just click on the check box of the desired criterion, after that, only the images that have this particular criterion will be shown in the image browser.

    You can check more than one criterion to search. Criteria inside a category are cumulative: the images that match any criteria will be shown. Criteria outside categories are exclusive, so the image has to be included in some criterion of both categories.

  4. Drag this slider to resize the image thumbnails.

  5. Each image thumbnail is composed of the image, a title with its file name and a subtitle with information. You can hide and show the titles from the main menu under view‚ÜíImage browser.

    To show or hide the image browser, click the central button in the Sidebars control in the toolbar. Alternatively, use the main menu or its shortcut.

    You can resize the image browser or collapse dragging its top edge.
