Exportar

Cómo exportar un álbum​

La forma más rápida de exportar tu álbum en Maqueteer es hacer click en el botón "Exportar" en la barra de herramientas. Alternativamente, puedes usar el menú principal Archivo > Exportar, o presionar Opción+Comando+E, para abrir el cuadro de diálogo de Exportación.

Nota: Para iniciar el proceso de exportación, todas las imágenes originales usadas en el álbum deben ser accesibles por Maqueteer. Si no es el caso, se mostrará un error informativo.

También se te advertirá si alguna de tus imágenes no tienen un perfil de color. Si ignoras esta advertencia, se asumirá que estas imágenes tienen el perfil de color sRGB.

  1. Puedes exportar el álbum completo, sólo el pliego actual en el lienzo, o los pliegos seleccionados en el navegador.
  2. Actualmente, Maqueteer soporta la exportación de tu álbum en los formatos JPEG, PDF, TIFF, PNG, HEIC, o PSD con capas.
  3. Puedes elegir entre sRGB, Adobe RGB 1898, ProPhoto o Display-P3.
  4. Ajusta al 100% para obtener el mejor resultado, pero si es para una vista previa o revisión de tu trabajo, el 80% es un buen compromiso entre el tamaño de los archivos y la calidad obtenida.
  5. El nivel de enfoque, el valor por defecto es medio.
  6. Te permite exportar tu álbum como páginas individuales en lugar de pliegos completos.
  7. Haz click si quieres redimensionar tu álbum de exportación para compartir, revisar, vista previa, etc.
  8. Puedes incluir marcas de revisión al exportar: número de página/pliego, número de imagen y nombre de archivo de imagen.

Cuando estés satisfecho con los ajustes, simplemente haz click en "siguiente..." y tras seleccionar el destino, el proceso de exportación comenzará.

Exportar con las versiones Pro vs. Básica​

Maqueteer tiene dos versiones disponibles. La versión Básica es completamente gratis, y tiene las mismas funcionalidades que la versión PRO. Pero la exportación tiene algunas limitaciones:

  • La resolución de exportación está limitada a 254 dpi.
  • El tamaño de "página" de exportación está limitado a 8 pulgadas de ancho o alto (20x20 cms). Por lo que el tamaño de tu pliego no puede ser mayor de 16x8 pulgadas (40x20 cms).

La versión Pro no tiene ninguna de estas limitaciones.

Exportar con un tamaño personalizado​

1.- Puedes especificar el nuevo tamaño estableciendo una nueva altura o anchura.

2.- Usa las unidades que quieras para configurar estas opciones.

3.- DPI (punto por pulgada) de resolución puede ser ajustado de 70 a 600.

4.- Si marcas esta casilla, el área de corte no será exportada. Útil para revisión, compartir, etc.

5.- Aquí puedes ver el tamaño de exportación final de los pliegos.
